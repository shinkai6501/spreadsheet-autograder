# Classroom スプレッドシート課題 自動採点

Google Classroomで「各生徒にコピーを作成」して配布したスプレッドシート課題を、提出物フォルダ単位で自動採点するGoogle Apps Scriptです。

Classroom APIは使用しません。ClassroomがGoogle Driveに作成した提出物フォルダを `DriveApp` で走査します。

## 処理の流れ

1. 先生が提出物フォルダURLを設定します。
2. 先生が模範解答スプレッドシートURLを設定します。
3. フォルダ内のGoogleスプレッドシートをすべて自動取得します。
4. ファイル名を生徒答案のインデックスとして使用します。
5. 模範解答と生徒答案の同名シート同士を比較します。
6. 模範解答に数式があるセルだけを採点します。
7. 全シートの数式とグラフを合算して10点満点にします。
8. 提出物フォルダ内に新しい採点結果スプレッドシートを作成します。

## 導入

1. 採点管理用のGoogleスプレッドシートを作成します。
2. `拡張機能 > Apps Script` を開きます。
3. `AutoGrader.gs` の内容を貼り付けて保存します。
4. 管理用スプレッドシートを再読み込みします。
5. `フォルダ自動採点 > 初期設定シートを作成` を実行します。

初回実行時は、スプレッドシートとGoogle Driveへのアクセス権限を承認してください。

## 設定

`設定` シートに以下を入力します。

- `提出物フォルダURL`
  - Classroomが課題用に作成したGoogle DriveフォルダのURL
- `模範解答スプレッドシートURL`
  - 正しい数式が入力済みのスプレッドシートURL
- `結果ファイル名`
  - 実際のファイル名には実行日時が追加されます
- `総点`
  - 初期値は10点
- `グラフ配点割合`
  - 模範解答にグラフがある場合、初期値では10点中3点をグラフへ配点
- `サブフォルダも検索`
  - TRUEなら配下のフォルダも再帰的に検索
- `採点対象外シート名`
  - 採点しない模範解答シートをカンマ区切りで指定
- `計算結果一致を正答扱い`
  - 初期値はFALSE。数式が異なれば、結果が同じでも0点
- `直打ちを正答扱い`
  - 数式ではなく計算結果を直接入力した答案を正答にする設定

数式を入力させる課題では、`直打ちを正答扱い = FALSE` を推奨します。

R1C1数式とグラフ種類が模範解答にすべて一致した答案は、合計10点になります。クラス平均を調整する相対評価は行いません。

## 実行

`フォルダ自動採点 > 提出フォルダを採点` を実行します。

採点前に対象セルを確認したい場合は、`模範解答から採点仕様を確認` を実行してください。管理用スプレッドシートの `採点仕様` シートに、対象シート名、セル、R1C1数式、配点が出力されます。

`提出フォルダを採点` は、表示されている `採点仕様` シートを正式な採点条件として使用します。行の削除・追加、シート名、セル、期待数式R1C1、グラフ種類、配点を手入力で修正できます。配点は比率として扱われ、合計が設定シートの総点になるよう自動調整されます。

`模範解答から採点仕様を確認` をもう一度実行した場合だけ、採点仕様シートは模範解答の内容で再生成され、手入力した修正が上書きされます。`提出フォルダを採点` を実行しても採点仕様シートは書き換えられません。

## 採点方法

模範解答スプレッドシート内の全シートを調べ、数式が入力されているセルを採点対象にします。

生徒答案に同名シートがある場合、同じセルを以下の順で判定します。

- `数式一致`: R1C1形式の数式が一致
- `計算結果一致（数式違い）`: 数式は異なるが計算結果が一致
- `直打ち`: 数式がなく、直接入力された値が計算結果と一致
- `空欄`: 提出セルが空欄
- `不一致`: 数式も計算結果も一致しない
- `シートなし`: 生徒答案に同名シートがない

配布時点から入力されている見出し、説明文、元データなどの値セルは点数に入りません。

## グラフ採点

模範解答に埋め込みグラフがある場合、同名シートにある生徒のグラフとグラフ種類だけを比較します。

グラフ種類が一致すれば、そのグラフの配点をすべて得ます。データ範囲、タイトル、軸タイトル、凡例、ヘッダー、色、書式、シート上の配置位置は採点しません。模範解答に複数のグラフがある場合は、同じ種類のグラフが必要数作成されているかを判定します。

## 結果スプレッドシート

採点結果は提出物フォルダ内に新規作成されます。

### 採点サマリ

1ファイルにつき1行で、以下を出力します。

- ファイル名
- 得点、満点、得点率
- 数式一致、計算結果一致、直打ち、空欄、不一致の件数
- グラフ一致、グラフ不一致の件数
- 不足しているシート名
- 提出ファイルURL

ファイル名がそのままインデックスになります。

### 採点詳細

各ファイル、シート、セルごとのR1C1数式と判定を出力します。

### 実行情報

提出物フォルダ、模範解答、対象シート、採点対象セル数などを記録します。

## 再実行

このプログラムが作成した結果スプレッドシートには識別用の説明を付けます。次回以降に同じ提出物フォルダを採点しても、以前の結果ファイルは答案として採点されません。

## 注意

- 実行者が提出物フォルダと全提出ファイルを閲覧できる必要があります。
- 模範解答と生徒答案で採点対象シート名を同じにしてください。
- 同名シートがない場合、そのシート内の採点対象セルは0点になります。
- 大人数かつ数式セルが非常に多い課題では、Apps Scriptの実行時間上限に達する場合があります。
