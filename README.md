# Google Apps Script 自動採点プログラム v2

`AutoGrader.gs` を管理用スプレッドシートの Apps Script に貼り付けて使います。

## v2で変えた点

- 生徒答案も、模範解答と同じように「どのシートを採点するか」を指定できます。
- 初期設定は `数式のみ` 採点です。あらかじめ配布した値セルは、原則として点数に入りません。
- 複数シートをまとめて採点し、スプレッドシート全体で100点満点にできます。
- 採点対象外の確認テストは `参考` として実行できます。結果には出ますが点数には入りません。
- 模範解答がない自由課題も、グラフ数、入力セル数、数式セル数、指定シート、特定テキストなどでチェックできます。

## 基本手順

1. 採点管理用スプレッドシートを作ります。
2. `拡張機能 > Apps Script` に `AutoGrader.gs` を貼り付けます。
3. 保存してスプレッドシートを再読み込みします。
4. `自動採点 > 初期シートを作成` を実行します。
5. `設定`、`採点対象シート`、`提出URL一覧` を入力します。
6. `自動採点 > 仕様生成と採点を実行` を実行します。

## シート指定

`設定` シートの `採点対象シート指定方法` で指定できます。

- `単一シート`: `模範解答シート名（単一シート用）` だけ採点
- `すべてのシート`: 模範解答ブック内の管理用シート以外を全部採点
- `シート範囲`: `シート範囲開始` から `シート範囲終了` までをまとめて採点
- `採点対象シート一覧`: `採点対象シート` シートの有効行を使う

`採点対象シート` に有効行がある場合は、一覧指定が優先されます。

## 生徒シート指定

`採点対象シート` シートで、行ごとに以下を指定します。

- `模範解答シート`: 模範解答側のシート名
- `生徒シート`: 生徒答案側のシート名

`生徒シート` が空欄なら、模範解答シートと同名の生徒シートを探します。シート名が違う課題では、ここに生徒側シート名を入れてください。

## 採点対象セル

`既定の採点対象セル` または `採点対象シート` の `採点対象セル` で指定します。

- `数式のみ`: 模範解答で数式が入っているセルだけ採点
- `数式+値`: 数式セルと値セルを採点
- `値のみ`: 値セルだけ採点
- `すべて`: 空欄や入力欄候補も含めて採点
- `明示範囲のみ`: `明示範囲A1` に書いた範囲だけ採点

数式穴埋め課題は `数式のみ` 推奨です。これで、配布時点で入っている見出しや固定値が点数に混ざりません。

## 複数シートで100点満点

`採点対象シート` の `配点` を空欄にすると、`設定` の `総点` から自動配分します。

例:

- Sheet1、Sheet2、Sheet3をすべて `採点`、配点空欄
- `総点` が100
- 各シートに均等配点

特定シートだけ重くしたい場合は、`配点` に数値を入れます。空欄の項目には残り点が自動配分されます。

## 参考チェック

`採点対象シート` または `自由課題チェック` の `区分` を `参考` にすると、点数には入れずに結果だけ出せます。

採点対象外のテスト、確認用シート、授業者が見たい補助チェックに使えます。

## 自由課題

模範解答がない課題では、`設定` の `模範解答を使う` を `FALSE` にして、`自由課題チェック` を使います。

対応チェック:

- `グラフ数以上`
- `入力セル数以上`
- `数式セル数以上`
- `シート数以上`
- `指定シートあり`
- `テキストを含む`

例:

- グラフを1つ以上作る: `グラフ数以上` / 対象 `すべて` / 条件値 `1`
- データを20セル以上入力する: `入力セル数以上` / 対象 `すべて` / 条件値 `20`
- 出典を書かせる: `テキストを含む` / 対象 `すべて` / 条件値 `出典`

## 出力

- `採点仕様`: 模範解答解析結果、採点対象セル、配点、自由課題チェック条件
- `採点結果`: 生徒ごと、シートごと、セル/チェックごとの詳細
- `採点サマリ`: 生徒ごとの得点、満点、得点率、判定集計

## 注意

- 初回実行時にスプレッドシートアクセス権限の承認が必要です。
- 提出スプレッドシートは、実行者が閲覧できる共有設定にしてください。
- 数式比較はR1C1形式で行います。
- `計算結果一致を正答扱い` と `直打ちを正答扱い` は、授業方針に合わせて切り替えてください。数式を埋める課題では `直打ちを正答扱い = FALSE` 推奨です。
